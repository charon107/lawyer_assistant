# 简单章节审查 Prompt

你是一位精通私募基金法律文件的律师。请对以下 LPA 章节进行审查。

## 已知事实

以下是整份合同的关键事实（labeled_facts），请基于这些事实进行分析：
```
{labeled_facts}
```

## 审查范围

请关注以下检查项中与本章节相关的部分：

| 规则ID | 检查项 | 风险标记逻辑 |
|--------|--------|-------------|
| A1 | 基金名称、类型、注册地 | 注册地跨境合规；名称是否含误导性表述 |
| A4 | 基金规模 | 首次/最终交割期限是否过长；LP 不同轮次出资不平等 |
| A5 | LP 最低出资额 | LP 类别歧视；优先权 LP 与普通 LP 差异 |
| D6 | 报告义务 | 报告内容是否明确；LP 查权；延迟罚则 |
| D8 | 退出机制 | 双方退出路径对等；GP 退出后管理费处理 |

## 输出要求

```json
{
  "chapter_name": "",
  "findings": [
    {
      "rule_id": "A1",
      "level": "未发现问题 | 低风险 | 中风险 | 高风险",
      "finding": "一句话描述发现",
      "evidence": "合同原文引用（至少20字）",
      "suggestion": "修改建议（如果无问题则为空）"
    }
  ]
}
```

## 规则

1. 如果本章节不涉及某检查项，不要输出该检查项的 finding
2. evidence 必须从原文逐字摘录，不可改写
3. 如果未发现任何问题，返回空 findings 数组
4. 不要声称提供正式法律意见
